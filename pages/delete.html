Delete a Resource with curl

To delete a resource on this server, use the following curl command in your terminal:

curl -X DELETE ip:port/path_to_file"

For example: curl -X DELETE localhost:400/pages/uploads/test.txt

Replace "Your/Requested/Path.resource" with the actual path to the resource you want to delete.

**Important:** Sending a DELETE request permanently removes a resource. Use it with caution!

Note: Html forms aren't compatible with sending delete requests. This is why we use curl.
